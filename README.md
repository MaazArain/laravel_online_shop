# laravel_online_shop

Laravel Online Shop
Laravel Online Shop is an eCommerce-based web application currently in development. This project is built using the Laravel framework and includes both an admin panel and a frontend website.

🔧 Features
Admin Panel:
Manage Categories

Manage Subcategories

Manage Brands

Manage Products

Full CRUD functionality for all entities

Frontend Website:
Display of all Products

Category and Subcategory Filtering

Add to Cart Functionality

View and Update Cart

Products dynamically loaded from the admin panel

🚧 Project Status:
This project is currently under active development. More features will be added soon.

🛠️ Tech Stack:
Laravel (PHP Framework)

Blade Templates (Frontend Templating)

MySQL (Database)

Bootstrap / CSS (Styling)
